# Setting-up-Ubuntu-Rpi-Properly
Troubleshooting info to set up Ubuntu 22.04.4 LTS (Jammy Jellyfish) on Rasbperry Pi for ROS2 (Humble) work and PX4 stack Via Telem2 Port on Pixhawk 6X Board
